# PP Mori — Tipografía Corporativa GreenTransit

Colocar aquí los archivos de fuente PP Mori con los siguientes nombres:

- `PPMori-ExtraLight.woff2` (peso 200)
- `PPMori-ExtraLight.woff`
- `PPMori-Regular.woff2` (peso 400)
- `PPMori-Regular.woff`
- `PPMori-SemiBold.woff2` (peso 600)
- `PPMori-SemiBold.woff`

PP Mori es una fuente comercial de Pangram Pangram Foundry. Asegúrate de tener la licencia web adecuada.
Mientras no estén disponibles los archivos, el sistema usará el fallback `'Source Sans 3'`.
